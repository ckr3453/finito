# 설명 마크다운 에디터

- phase: 5
- size: M

## 목표
- 할일 설명 입력 시 서식 툴바를 제공하여 마크다운을 직접 타이핑하지 않아도 되도록 개선
- 할일 상세 화면에서 마크다운을 렌더링하여 보기 좋게 표시

## 완료 기준
- [ ] `flutter_markdown` 패키지 추가
- [ ] 할일 등록/편집 화면 설명 필드 위에 서식 툴바 추가 (Bold, Italic, Strikethrough, Bullet list, Numbered list, Code)
- [ ] 툴바 버튼 클릭 시 선택된 텍스트에 마크다운 문법 자동 삽입 (미선택 시 커서 위치에 삽입)
- [ ] 할일 상세 화면에서 설명을 마크다운 렌더링으로 표시
- [ ] 기존 데이터 호환: 저장 포맷은 String 그대로 (DB 변경 없음)
